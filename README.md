React Tutorial TypeScript
====


[React Tutorial](http://facebook.github.io/react/docs/tutorial.html) をTypeScriptで書いてみる

## Description

## Setup

```
npm install -g dtsm gulp
npm install
dtsm install
```

```
cd server
npm install
```

## Usage

```
gulp
open http://localhost:3000/
node server/server.js
```
